Возможности

Всё что нужно для безопасного общения

End-to-End шифрование

Все сообщения зашифрованы на стороне отправителя и расшифровываются только на стороне получателя.

Локальный сервер

Данные никогда не уходят в облако. Сервер работает на одном из участников чата.

Защита паролем

Только люди с паролем могут подключиться и прочитать сообщения.

Простой интерфейс

Интуитивно понятный дизайн. Ничего лишнего — только общение.

Мультичат

Подключайтесь к существующим чатам или создавайте свои. Общайтесь с несколькими людьми одновременно.

Кроссплатформенность

Работает на Windows и Linux. Скачайте и запустите — никакой установки не требуется.

Как это работает

Три простых шага к безопасному общению

1

Настройте профиль

Задайте никнейм и пароль в боковой панели приложения.

2

Создайте или подключитесь

Нажмите Start Server чтобы создать чат, или Connect чтобы подключиться по IP и порту.

3

Общайтесь безопасно

Все сообщения шифруются автоматически. Сервер видит только зашифрованные данные.

Безопасность — наш приоритет

Encrit Messenger использует проверенные криптографические алгоритмы для защиты ваших сообщений.

  • Шифрование на базе PBKDF2 + HMAC-SHA256
  • Пароль никогда не передаётся по сети
  • Сервер не имеет доступа к расшифрованным сообщениям
  • Данные не покидают вашу локальную сеть
